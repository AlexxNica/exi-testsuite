Framework code
--------------

- Cache defeating java.io.InputStream implementation for use
  with in memory data source

- Network-based data sources and data sinks

- SAX processing encoding driver framework

- Typed Event API

- Event-based processing encoding and decoding framework

- Namespaces to schema source using catalog mapping

- Generation ordered list (MFUs) of strings and qualified names
  generated from schema and documents

Scripts
-------

- utilities to extract data from Japex report.xml files and convert to
  MS Office and ODF spreadsheet formats.
